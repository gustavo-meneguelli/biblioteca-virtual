# Biblioteca Virtual
Bem-vindo à Biblioteca Virtual! Este é um projeto em C# que simula uma biblioteca, permitindo que você navegue por livros, os compre e até mesmo os adicione ou remova do catálogo.

Funcionalidades Principais:
Navegue pelos Livros: Explore nosso catálogo e descubra uma variedade de livros disponíveis.
Adicione e Remova Livros: Seja um bibliotecário virtual e gerencie o acervo adicionando novos livros ou removendo os que não estão mais disponíveis.
Compre Livros: Escolha seus livros favoritos e compre-os com facilidade, usando dinheiro ou cartão de crédito.
Veja suas Compras: Revise suas compras anteriores e mantenha-se atualizado com sua coleção.

Como Usar:
Clone o repositório.
Abra o projeto em seu ambiente de desenvolvimento C#.
Execute o programa e siga as instruções simples do menu para começar a explorar e interagir com a biblioteca.
